## Detailed Pinout (Default Setup: ESP32-S3)

For the current build, the dual-bus master configuration is as follows:

### 1. Display (SPI2)
| Function | GPIO Pin |
| :--- | :--- |
| TFT_CS | 4 |
| TFT_RST | 5 |
| TFT_DC | 6 |
| TFT_MOSI | 7 |
| TFT_SCLK | 15 |

### 2. SD Card (SPI3)
| Function | GPIO Pin |
| :--- | :--- |
| SD_CS | 16 |
| SD_MOSI | 13 |
| SD_SCLK | 12 |
| SD_MISO | 17 |

*(Please note: SD_MISO is vital for the SPI3 bus connection.)*
